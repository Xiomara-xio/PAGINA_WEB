Semana 01

Clases y Actividades

Lenguaje Java: Java es un lenguaje de programación orientado a objetos y multiplataforma, creado por Sun Microsystems en 1995. Se utiliza ampliamente para desarrollar aplicaciones web y se puede ejecutar en numerosos dispositivos a nivel global.

IDE NetBeans: NetBeans es un entorno de desarrollo integrado (IDE) gratuito y de código abierto que se utiliza principalmente para el desarrollo de aplicaciones en Java. Permite crear distintos tipos de aplicaciones, incluyendo aplicaciones de escritorio, web y móviles, y está basado en Java, utilizando la biblioteca Swing para su interfaz gráfica.

Apache Maven: Es una herramienta que estandariza la configuración de un proyecto en todo su ciclo de vida, como por ejemplo en todas las fases de compilación y empaquetado y la instalación de mecanismos de distribución de librerías, para que puedan ser utilizadas por otros desarrolladores y equipos de desarrollo.

SEMANA01:

Realizar un programa que halle el área y perímetro de cuadrado, utilizando Lenguaje Java, el IDE NetBeans y el marco de gestión de proyectos de software Maven.
